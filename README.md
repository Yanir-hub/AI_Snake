How to Play

1. **Game Modes**
   - Single Player: Classic snake gameplay
   - VS AI: Challenge an AI opponent with adjustable difficulty

2. **Controls**
   - Use arrow keys (‚Üë, ‚Üì, ‚Üê, ‚Üí) to control your snake
   - Press Enter to start the game
   - Press ESC to return to menu

3. **Rules**
   - Eat red apples to grow longer and score points
   - Avoid hitting walls and snake bodies (including your own)
   - First to reach the target score wins!
   - If you crash, game over!

4. **Special Features**
   - Customizable snake and game colors
   - Adjustable game speed (Slow/Normal/Fast)
   - Optional power-ups that can freeze opponents
   - Three AI difficulty levels (Easy/Medium/Hard)

5. **Power-Ups** (when enabled)
   - Blue orbs appear randomly after eating apples
   - Collecting them freezes your opponent temporarily
   - Strategic power-up usage can help you win!
